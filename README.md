# Lecturas

A personal reader. Annotated adaptations of podcasts, videos, and articles worth reading more carefully than once. Each piece ships in **English** and **Spanish**.

**Live:** https://aauml.github.io/lecturas/

## Structure

```
.
├── index.html                              # Contents / cover page
└── articles/
    └── {slug}/
        ├── en/
        │   └── index.html                  # English version
        └── es/
            └── index.html                  # Spanish version
```

URL pattern:
- `aauml.github.io/lecturas/articles/{slug}/en/`
- `aauml.github.io/lecturas/articles/{slug}/es/`

Each article version includes a top-of-page language switcher to jump to the other language.
